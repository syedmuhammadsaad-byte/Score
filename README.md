Score: Training Language Models to Self-Correct via RLThis project implements the SCoRe (Self-Correction via Reinforcement Learning) algorithm, a training paradigm that enables Large Language Models (LLMs) to perform intrinsic self-correction.  About the ProjectTraditional LLMs often fail at self-correction when naively prompted, sometimes degrading in accuracy by as much as 11.2%. This project utilizes an online multi-turn Reinforcement Learning approach to teach models to:  Detect errors in their own reasoning without external feedback.  Correct specific mistakes through genuine self-reflection rather than behavior collapse.  Key FeaturesIntrinsic Self-Correction: The model receives zero correctness information (no gold answers or critic scores) between turns.  Two-Stage Training:Stage I: Decouples turn 1 and turn 2 distributions to prevent collapse.  Stage II: Uses joint optimization and reward shaping to bias the model toward genuine correction.  Robust Performance: Leverages multi-turn RL to achieve significant improvements in intrinsic self-correction deltas on benchmarks like MATH and HumanEval.  How It WorksThe model follows a specific meta-strategy:Turn 1: Generates an initial response based on the problem input.  Turn 2: Receives a generic correction prompt ("There might be an error...") and attempts to detect and fix any issues present in the first response.  ResultsMATH Dataset: Achieved a +15.6pp improvement in self-correction delta compared to the base model.  Code Generation: Showed strong generalization, achieving a +12.2pp improvement on HumanEval.
